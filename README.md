# Appointment_Booking_System
Nurbaev Shabdan

Description

This project is a simple appointment booking system developed in Java. It allows users to create, view, update, and delete appointments.

Objectives

- Practice Object-Oriented Programming (OOP)
- Implement CRUD operations
- Store and load data from files

Features

- Add appointment
- View appointments
- Update appointment
- Delete appointment
- Save data to file
- Load data from file
- Export data to CSV

OOP Concepts Used

- Encapsulation (private fields, getters/setters)
- Inheritance (Person → User, Specialist)
- Polymorphism (method overriding)

Technologies

- Java
- File handling (FileWriter, Scanner)

How to Run

1. Run Main.java
2. Use menu to manage appointments
